# Projeto Java - Gestão de Funcionários

Projeto desenvolvido em **Java puro** como exercício técnico para manipulação de lista de funcionários.

## Funcionalidades
 Operações:

- Cadastro de funcionários em lista
- Remoção de funcionário por nome
- Impressão dos dados em formato de tabela
- Aplicação de aumento salarial de 10%
- Agrupamento de funcionários por função
- Listagem de aniversariantes dos meses 10 e 12
- Identificação do funcionário com maior idade
- Ordenação alfabética
- Soma total dos salários
- Cálculo de salários mínimos por funcionário
